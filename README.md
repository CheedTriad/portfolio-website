# Chidiere Oluoma - Portfolio

Personal portfolio website built to showcase my projects, experience, and technical skills.
Focused on data analysis, machine learning, and the overlap between technology and business strategy.


<!-- View the demo [here](https://demo.magic-portfolio.com). -->

![My Portfolio](public/images/og/demo_homepage.jpg)

## About
I'm a final-year Computer Science student at De Montfort University, Leicester, on track for
First Class honours. My work sits at the intersection of data, business analysis, and
technology — I build in Python and Java, work with machine learning, and care about
what the output actually means for the people using it.

This site covers my projects, work history, and technical background.

## Built With

- [Next.js](https://nextjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Once UI](https://once-ui.com)
- [MDX](https://mdxjs.com)
- Deployed via [Vercel](https://vercel.com)
